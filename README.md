## Generic Effects System (通用效果系统)

### Introduction(介绍)

This is an Unreal Engine 5 code plugin which helps you to:

    1.Play any sfx/vfx based on different context, for any things that requires feedback.
    2.Data-driven setup of sfx/vfx that can be played based on gameplay tags(contexts).
    3.Automatic footsteps sfx/vfx setup via animation modifers.

这是一个虚幻引擎5代码插件，可以帮助你完成如下内容：

    1.在不同情境下播放特效和声效，适用于任何需要反馈的地方。
    2.根据游戏标签（上下文），以数据为驱动设置可播放的 声效/特效。
    3.自动化检测脚步并配置脚步声效/特效的播放。
    

This module is mainly used for footstep vfx/sfx， But you can also use this plugin to play sfx/vfx based on other gameplay event such as melee hit,
bullet impacts etc.

此插件主要用于脚步系统，但是你完全可以可以基于游戏中的其他事件进行声效/特效的播放，比如近战碰撞效果，或者远程子弹碰撞效果。
